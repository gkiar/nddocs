LONI PipelineÂ¶

The LONI Pipeline tool sits on top of the system scheduler (Son of Grid Engine, for instance), and allows the user to easily string together modules (i.e. single commandline executable calls) to form workflows. This environment is our choice tool for creating pipelines, as it allows us to easily pipe outputs from one module to the input of another.

An example LONI Module from one of our projects, m2g, can be used as a reference when creating your own modules.

If you are looking for more guidance when creating modules, checkout LONI’s tutorial videos.
