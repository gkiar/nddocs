Open Connectome Project Documentation

This site provides an overview to the services currently offered by the Open Connectome Project (OCP). We provide visualization, data access, query, and analytics capabilities for neuroscientists, computer scientists, and the broader scientific community. Our mission is to enhance big data neuroscience.
